#Create Module
- Buat Repository pada GITHUB
- Tambah Folder Pada Windows Explorer 
    * folder app_say_hello = folder aplikasi yang nanti yang akan di arahkan ke modulenya
    * folder go_say_hello = folder modulenya
- commit di GITHUB
- menambahkan tag (git tag v1.0.0)
- menambah dependency =  (go mod init github.com/widiarta112/app-say-hello) pada app_say_hello
- memasukan module pada app = (go get github.com/widiarta112/go-say-hello) pada app_say_hello
